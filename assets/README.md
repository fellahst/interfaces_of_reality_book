# Cover Images

This folder contains the front cover and back cover images for the book.

## Files

- `front-cover.jpg` (or `.png`) - Front cover image
- `back-cover.jpg` (or `.png`) - Back cover image

## Image Requirements

### Front Cover
- Recommended dimensions: 1600 x 2560 pixels (for Kindle/ebook)
- Or: 2550 x 3300 pixels (for print)
- Format: JPG or PNG
- Resolution: 300 DPI for print, 72 DPI for ebook

### Back Cover
- Recommended dimensions: Same as front cover
- Format: JPG or PNG
- Resolution: 300 DPI for print, 72 DPI for ebook

## Usage

These images can be used when:
- Publishing to Amazon KDP (Kindle Direct Publishing)
- Publishing to Leanpub
- Creating print versions
- Generating EPUB/MOBI files
